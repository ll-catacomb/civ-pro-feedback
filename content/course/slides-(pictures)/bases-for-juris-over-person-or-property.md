# Bases of Jurisdiction Over Person or Property

<!-- page: 1 -->

> **Diagram type:** Three-column reference table comparing in rem, quasi-in-rem, and in personam jurisdiction.

## Text Extracted from Document

### In rem

- Step one: court "attaches" a piece of property "in" its geographic jurisdiction
- Step two: in the same case, court adjudicates ownership over that piece of property as against the world
- Lawsuit is against the piece of property
- Claimant ordinarily claims some ownership interest (*e.g.*, lien)
- Funny names: "In re The Nautilus," "In re 221b Baker Street, London," "In re #4 Privet Drive"

### Quasi-in-rem

- Step one: court "attaches" a piece of property "in" its geographic jurisdiction
- Step two: pla proves that owner of property wronged pla in some way (usually unrelated to piece of property seized)
- Step three: if pla prevails, court sells property, gives pla the money up to the amount of the claim, gives defendant leftovers
- Pla's recovery limited to value of the property
- Used when defendant hard to reach

### In personam

Lawsuit 1, State A, usually more convenient for pla: pla sues defendant; defendant can either:

- Appear, litigate whether IPJ in State A
  - If def wins, case dismissed
  - If def loses on IPJ, litigate on merits, later appeal in State A
- Default

If pla prevails, State A court issues judgment; pla then finds def property in State B (sometimes A=B)

Lawsuit two, in State B, usually more convenient for defendant: pla requests State B court seize property, sell it to satisfy judgment

- If def appeared in lawsuit 1, cannot object to IPJ (usually loses)
- If def defaulted in lawsuit 1, can litigate whether State A court had IPJ (in State B!)
