# Offensive and Defensive Collateral Estoppel

<!-- page: 1 -->

> **Diagram type:** Two side-by-side arrow diagrams contrasting offensive and defensive use of collateral estoppel across two lawsuits.

## Text Extracted from Document

### Offensive Collateral Estoppel

**Lawsuit 1**
- P1 → D
- D loses on an issue

**Lawsuit 2**
- P2 → D
- P2 seeks to collaterally estop D

New plaintiff uses a prior loss against the defendant. Goal: leverage a prior win against the same defendant. **Discretionary** — court checks fairness (see *Parklane Hosiery* factors).

### Defensive Collateral Estoppel

**Lawsuit 1**
- P → D1
- P loses on an issue

**Lawsuit 2**
- P → D2
- D2 seeks to collaterally estop P

New defendant uses a prior loss against the plaintiff. Goal: shut down a repeat plaintiff. **Generally allowed** once elements are met.
