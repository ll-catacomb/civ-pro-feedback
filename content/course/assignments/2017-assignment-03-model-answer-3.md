## Assignment 3 — Civil Procedure 2, 2017

### Question 1: Motion to Dismiss

**Preliminary: SMJ, IPJ, Venue**

Courts should dismiss for lack of SMJ. Here, § 1332 provides SMJ: Aberforth ("AD") is a Maine domiciliary and Ginny ("GW") is a Nebraska resident/domiciliary, and the AIC of $150k exceeds $75k. Ginny was personally served in her home state, establishing personal jurisdiction. Venue of the transferor court was properly laid as Nebraska is GW's state of residence (§ 1391).

**Limitations Period: Transfer Analysis and VCOL**

HCOL is moot because both Maine and Nebraska have adopted the misconduct rule and their differences in SOL length cannot matter as to which state's rule applies on that specific issue.

| | SOL Start Date | SOL Length |
|---|---|---|
| Federal | Discovery Rule (FCL): 4/2017 | 1 year (Statute) |
| Maine | Misconduct Rule: 3/15/2016 | 3 years |
| Nebraska | Misconduct Rule: 3/15/2016 | 2 years |

Because the suit was filed 8/12/2017, the only circumstance where the limitations period could result in dismissal is if we mix and match: using either state's SOL start date (misconduct rule) with the federal 1-year SOL. Under all other combinations, AD filed in time.

**Transfer/Happy Court:** There is no happy court/transfer/HCOL issue. Under either Nebraska or Maine law (limitations period length and/or start date), the result is the same: the lawsuit is timely if we use either state's law, and untimely only if we mix the federal limitations period with either state's start-date rule. Nebraska was a happy court (Ginny domiciled and personally served there, FRCP 4(k)(1)(A), § 1391(b)(1)), so assume Nebraska law applies.

**VCOL:** There are two *Erie* issues: the applicability of the federal common law discovery rule for start dates, and the FLPA 1-year SOL.

**SOL Length:**

| | If Nebraska's Misconduct Rule Applies | If Fed Discovery Rule Applies |
|---|---|---|
| *York*: apply state law if outcomes vary? | Outcomes vary → apply Nebraska's 2-year SOL | Outcomes don't vary → can apply Fed 1-year SOL |
| *Hanna* Majority (statutory): apply federal statutes where valid and applicable | Apply Federal 1-year SOL — valid (Article III ambit) and applicable (FLPA specifically addresses this fact pattern) | Apply Federal 1-year SOL — valid and applicable |
| *Harlan*: apply state law if primary conduct would be affected | Apply Federal 1-year SOL — the SOL length wouldn't drive pre-litigation behavior, unless it affects car insurance companies | Apply Federal 1-year SOL — same reasoning |

Given the above analysis, it seems likely that the Federal 1-year SOL would apply.

**SOL Start Date:**

| | If Nebraska's 2-Year SOL Applies | If Fed 1-Year SOL Applies (likely) |
|---|---|---|
| *York*: apply state law if outcomes vary? | Outcomes don't vary → can apply Federal Discovery Rule | Outcomes vary → apply Nebraska's Misconduct Rule |
| *Hanna* Majority (FCL): apply state law if doing so serves twin aims of *Erie* | Apply Nebraska's Misconduct Rule — discourages forum shopping and inequitable administration by aligning outcomes across fora | Apply Nebraska's Misconduct Rule — same rationale |
| *Harlan*: apply state law if primary conduct would be affected | Apply Federal Discovery Rule — when SOL begins wouldn't drive pre-litigation behavior (unless insurance companies) | Apply Federal Discovery Rule — same reasoning |

Given the above, the *Erie* analysis suggests adopting the state's misconduct rule — but this is "Franken-law." Mixing the state's SOL start date with the federal 1-year SOL would result in a hybrid limitations period that conforms to neither state nor federal law. We should not recommend dismissal here.

**Failure to State a Claim**

Both states have adopted *Twiqbal* plausibility pleading. The analysis requires (1) striking conclusory statements from the pleading, and (2) assessing the remaining preliminary facts to ask whether they state a plausible case.

After striking the conclusory "negligence" language, the remaining preliminary facts are that GW was driving while intoxicated and collided with AD, causing him injury. Under both states' negligence tort theories, these facts if true satisfy the three elements and state a plausible case. We should not dismiss for failure to state a claim.

### Question 2: Issue Preclusion Against Harry

Issue preclusion requires: same issue; actually litigated; necessary to the decision; and mutuality/party identity.

- **Same issue:** Yes — who was the negligent party in the accident: who was driving while intoxicated.
- **Actually litigated:** Yes — specific jury findings via special interrogatories.
- **Necessary to decision:** Yes — GW's verdict depended on the jury finding it was Harry, not Ginny, who was driving.
- **Mutuality:** Depends on forum. Federal courts allow non-mutual offensive collateral estoppel (NMOCE) in limited situations where the defendant, who lost in L1, is precluded from re-litigating an issue (the posture is that the plaintiff changes from L1 to L2; the defendant is constant). Here, Harry was not a party in L1, so NMOCE would not be allowed.

But none of the above really matters: Harry (the potentially estopped party) was not a defendant in L1 and did not previously litigate the issue. It does not matter whether Nebraska, Maine, or federal law applies — estopping a non-party who was not in privity with a party would violate Due Process.
